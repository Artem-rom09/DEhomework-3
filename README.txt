# Проєкт Міграції Даних MySQL у MongoDB

Цей проєкт містить скрипти для вилучення даних про користувачів, рекламні покази та кліки з реляційної бази даних MySQL, трансформації їх у вкладену структуру документів та завантаження в базу даних MongoDB. Також включає аналітичні запити до даних у MongoDB.

---

## Структура Проєкту

Проєкт складається з таких основних файлів:

* `docker-compose.yml`: Конфігурація для запуску MongoDB у Docker.
* `migrate_data.py`: Основний скрипт для підключення до MySQL, вилучення, трансформації та завантаження даних у MongoDB.
* `run_queries.py`: Скрипт, що містить аналітичні запити до даних, завантажених у MongoDB.
* `README.md`: Цей файл з описом проєкту.

---

## Вимоги

Для запуску проєкту вам знадобляться:

* **Docker** та **Docker Compose** (встановлені та запущені).
* **Python 3.x** (встановлений на вашій машині).
* **Доступ до бази даних MySQL** з відповідними таблицями (`Users`, `Impressions`, `Campaigns`, `Advertisers`, `Clicks`, `UserInterests`, `Interests`).

---
